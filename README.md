# 🚨 Smart Fraud Transaction Dashboard

An interactive Power BI dashboard designed to analyze financial transaction data and identify fraud patterns using data visualization, Power Query, and DAX.

---

## 📌 Project Overview

Financial fraud is one of the biggest challenges faced by organizations today. This dashboard transforms raw transaction data into meaningful insights that help identify suspicious activities, monitor fraud trends, and support data-driven decision-making.

The dashboard provides interactive visualizations, KPIs, and filters that enable users to explore transaction behavior and detect fraudulent activities efficiently.

---

## 🎯 Objectives

- Analyze transaction data for fraud detection
- Build an interactive Power BI dashboard
- Visualize important KPIs
- Identify fraud patterns and trends
- Improve business decision-making using data

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CSV Dataset
- Data Modeling

---

## 📂 Dataset

Dataset: Synthetic Fraud Transaction Dataset

The dataset contains information such as:

- Transaction ID
- Transaction Amount
- Transaction Type
- Fraud Status
- Date & Time
- Customer Information
- Merchant Details

## 📊 Dashboard Features

- KPI Cards
- Fraud Transaction Analysis
- Legitimate vs Fraud Comparison
- Transaction Trend Analysis
- Category-wise Analysis
- Interactive Filters & Slicers
- Drill-down Reports

## ⚙️ Data Preprocessing

The following preprocessing steps were performed using Power Query:

- Removed duplicate records
- Handled missing values
- Converted data types
- Renamed columns
- Filtered unnecessary data
